# StreamSphere                                                                                                            
 It is a video streaming platform.                                                                                                  
 🔧 Tech Stack:                                          
 Frontend: React (Vite) + CSS                                
 Backend: Spring Boot (Microservices)                                    
 Database: MySQL (User &amp; Subscription Data) + MongoDB (AI Recommendations)                          
 Cloud: AWS (S3 for Video Storage, EC2 for Hosting, MediaConvert for Streaming)
